Proxy renewal with LCG-CE [Tags] case1
External Log ${TEST NAME}
Set Long Jdl ${util.JDLFILE}
Set Destination CE ${util.JDLFILE} 2119/jobmanager
Add Jdl Attribute MyProxyServer ${util.MYPROXYSERVER}
Log Info Submit a job with a short proxy
Set Proxy ${util.PROXY} 00:14
${jobid}= Submit Job ${util.JDLFILE}
Log Info Check if the job has been submitted with the right proxy
Check Job Proxy Duration ${util} ${jobid}
# We need now a long proxy to check job status
Set Proxy ${util.PROXY} 12:00
Wait Until Job Finishes ${jobid}
${final_status}= Get Job Status ${jobid}
Run Keyword if '${final_status}'=='Aborted' Check Failed Reason ${util}
... ${jobid}
Run Keyword if '${final_status}'=='Aborted' Fail Job's final status is Aborted
Run Keyword if '${final_status}'=='Cancelled' Fail Job's final status is Cancelled
Run Keyword if '${final_status}'=='Done (Failed)' Fail Job's final status is Done (Failed)
Log Info Test OK
