Shallow Resubmission [Tags] case1

${jobid}= Submit Job ${util.JDLFILE}
Wait Until Job Finishes ${jobid}
${final_status}= Get Job Status ${jobid}
Should Be Equal As Strings ${final_status} Aborted Something goes wrong, job final status is ${final_status}
${output}= Run Command glite-wms-job-logging-info --event RESUBMISSION ${jobid}
Log Check the number of resubmissions
${shallow_lines}= Get Lines Containing String ${output} SHALLOW
${shallow_count}= Get Line Count ${shallow_lines}
${target_count} Convert To Integer 2
Should Be Equal ${shallow_count} ${target_count} Job ${jobid} hasn't be correctly resubmitted. Find SHALLOW ${shallow_count} times.
